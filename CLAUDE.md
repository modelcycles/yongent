# yongent

## yt-dlp사용
- 막힘없이 안전하게 사용할 수 있는 방법 필요
- 쿠키를 직접 export 하여 업로드 하는 방식으로 사용

## 파이프라인
### 키워드 입력 시 작업 순서
#### 멜론 검색 파트 https://www.melon.com/ 에서 검색색
1. 멜론 검색 후 메타데이터 추출 (모두 합쳐 md파일로 export)
    1. 곡명
    2. 아티스트명
    3. 앨범명
    4. 발매일
    5. 작곡가
    6. 작사가
    7. 가사
    8. 추출한 멜론 사이트 url
2. 멜론 검색에서 나온 해당 곡에 대한 앨범 이미지 파일 다운로드
#### 유튜브 음원 다운로드 파트 https://www.youtube.com/ 에서 검색
1. 유튜브 검색 후 다운로드
    1. 원본 음원 mp3 형식
    2. 처음부터 60초 까지 분량의 편집 음원 (55-60구간은 페이드 아웃) mp3 형식

### 파일명 규칙
> 첫 입력시 url이 아닌 `아티스트명-곡명` 으로 입력 했다면 사용자가 입력한 그대로 파일명으로 가져갈것것
- 메타데이터 : `아티스트명-곡명(Meta).md`
- 앨범 이미지 : `아티스트명-앨범명.jpg` 아니면 `아디스트명-앨범명.png`
- 원본 음원 : `아티스트명-곡명.mp3`
- 편집 음원 : `아티스트명-곡명.mp3`
